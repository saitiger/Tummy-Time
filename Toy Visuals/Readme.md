## Installation & Setup
1. Clone the repository:
   ```sh
   git clone https://github.com/saitiger/Tummy-Time.git
   cd toy-visuals
   ```
2. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
3. Run the Streamlit app:
   ```sh
   streamlit run Home.py
   ```
